= Workbook: Fix AsciiDoc Admonitions
:toc: preamble
:icons: font

Admonitions render unstyled here — a bare table with the word "Tip" instead of an icon — because
`icons: font` is unset and nothing styles `.admonitionblock` (jekyll-asciidoc ships no CSS, and
Minimal Mistakes doesn't style it either). The fix is site-wide: config + one CSS file + one
`<link>`. No article edits.

== Step 2 — Write the stylesheet

Writes `site/assets/css/asciidoc.css` — the school's tested file, byte-for-byte. It is based on
Asciidoctor 2.0.26's `.admonitionblock` rules
(https://github.com/asciidoctor/asciidoctor/blob/v2.0.26/data/stylesheets/asciidoctor-default.css)
plus the two Minimal-Mistakes table fixes (`display:table`, `td{border:0}`). The script's header
comment claims "matches upstream exactly" — it does not: it also adds a `margin`, drops
`word-wrap:anywhere`, italicises the caption title, and omits two icon-off rules.
