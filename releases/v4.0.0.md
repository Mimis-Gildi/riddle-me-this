# Release v4.0.0

Major version bump. Single generic resume replaces four specialized variants.
Repository cleaned of legacy infrastructure accumulated over 3 years.

## ✨ What's New

- **Single resume**: `VadimKuhay-Resume.adoc` replaces OnEngineering, OnCreativity, OnLeadership, OnCore.
  Fragment-based assembly (hero, sub-hero, products, skills, experience) for maintainability.
- **Products and Platforms section**: new resume section showcasing highest-risk endeavors with outcomes.
- All resume references updated across README, site index, and maintainer page.

## 🧹 Removed

- **4 resume variants** (OnEngineering, OnCreativity, OnLeadership, OnCore) -- consolidated into one.
- **Saga export infrastructure** -- export script, ShellSpec tests, bootstrap helpers, test fixtures.
- **ShellSpec** -- framework, config, Makefile, helpers. No shell tests remain.
- **New user greeting workflow** -- `custom-new-user-greeting.yml` removed.
- **Conda environments** -- `.conda/` directory removed.
- **Kroki** -- Docker Compose files for local diagram rendering removed.
- **Orphaned assets** -- `resources/images/`, `resources/fragments/`, resume symlinks.
- **buildSrc** -- custom Gradle plugins (dependency processing, toolchain verification) removed.
- **tracks/** -- marketing drafts, consulting track, prospecting, code labs placeholders.
- **Release history** -- pre-v3.19.0 release notes cleaned out. Old releases deleted from GitHub.
- **Snyk workflow** reference removed from README badges.

## 🔬 Internal

- `.sdkmanrc` upgraded: Gradle 9.4.1, Java 21.0.10-tem, Kotlin 2.3.20.
- `CLAUDE.md` trimmed to project essentials -- removed stale counts, version numbers, memory hierarchy docs.
- `.codacy.yaml` cleaned of `.saga-notes/` exclude.
- `.github/labeler.yml` cleaned of `.saga-notes/` path.
- `update-version-references` action author updated.
- `build.gradle.kts` simplified.
- `.gitignore` simplified.
- Runner introspect workflow updated.
