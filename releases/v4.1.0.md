# Release v4.1.0

CI/CD infrastructure externalized to fluffle. Community governance defaulted to org-level policies.

## ✨ What's New

- **Community governance**: Defaulted to org-level Code of Conduct, Contributing guide, and Security policy
  via GitHub's community health files. Added project-specific `CONTRIBUTING-HERE.adoc` and `SECURITY-HERE.md`.

## 🧹 Removed

- **Local CI/CD actions**: `introspect-runner` (355 lines), `detect-runner-type` (41 lines) -- logic moved to
  `Mimis-Gildi/fluffle` shared workflows. Local workflows are now thin callers.
- **Local prune infrastructure**: `github-actions-prune.yml`, `github-cache-prune.yml`, and both shell scripts
  (`clean-workflow-runs.sh`, `clean-workflow-run-caches.sh`) -- replaced by fluffle `actions-prune.yml`.
- **Local PR labeler**: `custom-pr-labels.yml` -- replaced by fluffle `labeler.yml`.
- **Local runner introspect workflow**: `custom-runner-introspect.yml` (102 lines) -- replaced by
  `runner-introspect.yml` calling fluffle's `runner-introspector.yml`.
- **Issue and PR templates**: `add-content.md`, `remove-content.md`, `valuable-idea.md`, `publish-article-template.md`.
- **Local governance files**: `CODE_OF_CONDUCT.adoc`, `CODE_OF_CONDUCT.md`, `CONTRIBUTING.adoc`.

## 🔬 Internal

- Site copyright updated to 2022-2026, v4.1.0.
- `ruby/setup-ruby` updated to v1.300.0.
- `gradle/actions/setup-gradle` updated to v6.1.0.
- Runner introspect workflow renamed to "War Canary."
